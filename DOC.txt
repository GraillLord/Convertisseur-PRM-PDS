//Pour la version 1.6 ou java 6:
//(javafx n'étant pas inclu dans le sdk 1.6, il faut télécharger javafx 2.2)

*cas IDE:

-Ouvrir Main.java dans un projet eclipse/netbeans (créer new project, new java class et copier/coller le code source)
-importer la library javafx 2.2 pour java 1.6 (version 32 et 64 bit) dans eclipse/netbeans
compiler et créer le jar

*cas Manuel (sans IDE):

-ouvrir Main.java dans un editeur de texte (notePad par exemple)
-télécharger library javafx 2.2 pour java 1.6 (version 32 et 64 bit)
-créer un fichier "x86", dans lequel se trouvera le contenu du fichier "bin" et "lib" de javafx 2.2
-javac -cp "\Program Files (x86)\Oracle\JavaFX 2.2 RunTime\lib\jfxrt.jar" Main.java 	pour créer les .class
-java -cp .;"\Program Files (x86)\Oracle\JavaFX 2.2 RunTime\lib\jfxrt.jar" Main 	pour tester le programme
-jar cf myJar.jar myClass.class 	pour creer le jar
-jar -jar myJar.jar 	pour executer le jar
-Attention ne pas oublier de creer un manifest dans lequel on précise le classpath: jfxrt.jar et le mainclass: Main
-ajouter le jar dans le fichier "x86"
